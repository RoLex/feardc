Favorite Hub Properties

Hub

Name
Specifies what name the hub has.
Address
The address for the hub.
Description
Description of the hub.
Encoding
A special encoding to be used for this particular hub. Note that this setting has effect on NMDC hubs and the dropdown list may not appear if a non-NMDC hub address has been specified.

Identification (leave blank for defaults)

Nick
If you would like to use a special nick for a specific hub. Leave this blank for default.
Password
If you have a password attached together with your nick. Used for operators or other registered people.
Description
If you would like to use a special user description (shown in the user list) for a specific hub. Leave this blank for default.
Email
If you would like to use a special email (shown in the user list) for a specific hub. Leave this blank for default.
IPv4
If you would like to use a special external IPv4 address for a specific hub. Useful to log to local & remote hubs at the same time. Leave this blank for default.
IPv6
If you would like to use a special external IPv6 address for a specific hub. Useful to log to local & remote hubs at the same time. Leave this blank for default.
Group
Select a favorite hub group to be associated to the hub.
More information about the use of favorite hub groups can be found in the help topics associated to the Favorite hub groups dialog.
Manage groups
Open the Favorite hub groups dialog where you can add, edit, remove groups.
More information about the use of favorite hub groups can be found in the help topics associated to the Favorite hub groups dialog.
