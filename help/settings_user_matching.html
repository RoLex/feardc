User matching definitions

List of current user matching definitions.
Every user will be matched against configured user matching definitions to find properties to apply (such as chat ignoring, custom colors...). User matching definitions are browsed from the top of the list to the bottom; therefore, when 2 user matching definitions can be applied to the same user and affect the same property, the top-most definition takes precedence.

Add
Open a User matching definition dialog to create a new user matching definition and add it to the list.
Edit
Open the currently selected user matching definition in a User matching definition dialog in order to edit its properties.
Move Up
Move the currently selected user matching definitions upwards in the list.
Move Down
Move the currently selected user matching definitions downwards in the list.
Remove
Remove the currently selected user matching definitions.
