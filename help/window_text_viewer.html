Text viewer

This viewer displays the contents of a file as text. The name of the file is indicated in the tab title; the full path to the file is specified in the status bar.

Context menu commands

Above the basic text selection and copy to clipboard commands the following search related commands will appear in the menu:

Change the text viewer font

Press this button to change the font used by text viewers. A monospace font such as Courier New or Lucida Console is commonly required to properly view the ASCII_art contained in some text files.
