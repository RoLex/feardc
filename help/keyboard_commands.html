Keyboard shortcuts

This page enables you to see which special keyboard shortcuts exist in DC++. Note that standard Windows keyboard shortcuts are also fully supported throughout the program's graphical interface.

F1
Opens the help file.
(Shift +) Tab
Completes nicks when writing a chat message. Otherwise, moves the focus to the next (or previous, if shift is pressed) control.
Ctrl + (Shift +) Tab
Changes the main tab view to the next (or previous, if shift is pressed) window. In settings, switches to the next (or previous, if shift is pressed) settings page.
Shift
Pressing shift during program startup will disable auto-connection to hubs.
Ctrl + W
Ctrl + F4
Close the current window.
Ctrl + 0
Toggle the permanent visibility of the main menu bar. When it is hidden, the menu bar can be temporarily brought back into view with standard Windows shortcuts (Alt or F10).
Ctrl + 1
Enable or disable the toolbar.
Ctrl + 2
Enable or disable the transfer view.
Ctrl + 3
Enable or disable the status bar.
Ctrl + D
Open the Download Queue window.
F5
Ctrl + E
Refresh file list.
F3
Ctrl + F
In-place search within the chat of Hub and Private Message windows and in File List windows.
F3 continues the previous search or initiates one if no search term has yet been defined. On the other hand, Ctrl + F always initiates a new search.
Ctrl + G
Quick connect to a group of favorite hubs.
Ctrl + H
Open the Favorite Hubs window.
Ctrl + N
Open the Notepad window.
Ctrl + L
Open file list.
Ctrl + O
Open the System Log window.
Ctrl + P
Open the Public Hubs window.
Ctrl + Q
Quick connect to a hub.
Ctrl + R (in Hub windows)
Reconnect.
Ctrl + S
Open the Search window.
Ctrl + U
Open the Users window.
Ctrl + F3
Open the Settings dialog.
Ctrl and + (on the numeric keyboard)
If certain columns, e.g. those in the user list, search and the transfer view are not visible, pressing Ctrl and + (on the numeric keyboard) will resize the columns to the width necessary to accommodate the widest text within.
Ctrl + up arrow (in Hub and Private Message windows)
Move up in chat messages.
Ctrl + down arrow (in Hub and Private Message windows)
Move down in chat messages.
Alt + C (in Hub and Private Message windows)
Focus the chat part of the window.
Alt + G (in Hub windows)
Get the file lists of the selected users.
Alt + M (in Hub and Private Message windows)
Focus the message writing box of the window.
Alt + P (in Hub windows)
Open Private Message windows for the selected users.
Alt + S (in Hub and Private Message windows)
Send chat message.
Alt + U (in Hub windows)
Focus the user list.
Escape (in Hub and Private Message windows)
Focus the message writing box of the window.
Cancel the current chat search if there is any.
Alt + I (in the Public Hubs and Users windows)
Set focus on the filter input box.
+ (on the numeric keyboard)
In the download queue, pressing + (on the numeric keyboard) sets a higher priority on a download.
- (on the numeric keyboard)
In the download queue, pressing - (on the numeric keyboard) sets a lower priority on a download.
