Chat Commands

Main Chat

/join <hub-ip>
Joins <hub-ip>. See also Open new window when using /join.
/showjoins
Toggles the displaying of users joining the hub. This only takes effect for freshly-arriving users.
/favshowjoins
Toggles the displaying of favorite users joining the hub. Does not require /showjoins to be enabled.
/userlist
Toggles visibility of the list of users for the current hub.
/connection
/conn
Displays the connectivity status information, auto detected or manually chosen connection mode, IP and ports that DC++ is currently using for connections with all users.
/favorite
/fav
Adds the current hub (along with your nickname and password, if used) to the list of Favorite Hubs.
/removefavorite
/removefav
Removes the current hub from the list of Favorite Hubs.
/pm <user> [message]
Opens a private message window to the user, and optionally sends the message, if one was specified.
/getlist <user>
Adds the user's list to the Download Queue.
/ignore <user>
Adds a user matching definition (or modifies an existing one, if possible) to ignore chat messages from the specified user.
/unignore <user>
Adds a user matching definition (or modifies an existing one, if possible) to stop ignoring chat messages from the specified user.
/topic
Prints the current hub's topic. Useful if you want to copy the topic or it contains a link you'd like to easily open.

Private Message

/direct
/encrypted
Starts a direct encrypted communication channel to avoid spying on your private messages.
/getlist
Adds the current user's list to the Download Queue.
/grant
Grants the remote user a slot. Once they connect, or if they don't connect in 10 minutes, the granted slot is removed.
/favorite
/fav
Adds the current user to the list of Favorite Users.
/ignore
Adds a user matching definition (or modifies an existing one, if possible) to ignore chat messages from the current user.
/unignore
Adds a user matching definition (or modifies an existing one, if possible) to stop ignoring chat messages from the current user.
/lastmessage
Lists the date and time when the last message was sent or received.

Both

/refresh
Manually refreshes DC++'s share list by going through the shared directories and adding new folders and files. DC++ automatically refreshes once an hour by default, and also refreshes after the list of shared directories is changed.
/slots <number>
Sets the current number of upload slots to the number you specify. If this is less than the current number of slots, no uploads are cancelled.
/dslots <number>
Sets the current number of download slots to the number you specify. If this is less than the current number of slots, no downloads are cancelled.
/search <string>
Opens a new search window with the specified search string. It does not automatically send the search.
/close
Closes the current window.
/clear [lines to keep]
Clears the current window of all text. Optionally, you can specify how many of the latest (most recent) lines should be kept.
/dc++
Sends a random DC++ advertising message to the chat, including a link to the DC++ homepage and the version number.
/away [message]
Sets Away status. New private message windows will be responded to, once, with the message you specified, or the default away message configured in the Personal information settings page.
/back
Un-sets Away status.
/d <search string>
Launches your default web browser to the DuckDuckGo search engine with the specified search.
/g <search string>
Launches your default web browser to the Google search engine with the specified search.
/imdb <imdb query>
Launches your default web browser to the Internet Movie Database (imdb) with the specified query.
/rebuild
Rebuilds the HashIndex.xml and HashData.dat files, removing entries to files that are no longer shared, or old hashes for files that have since changed. This runs in the main DC++ thread, so the interface will freeze until the rebuild is finished.
/log <status, system, downloads, uploads>
If no parameter is specified, it launches the log for the hub or private chat with the associated application in Windows. If one of the parameters is specified it opens that log file. The status log is available only in the hub frame.
/help
Displays available commands. (The ones listed on this page.) Optionally, you can specify "brief" to have a brief listing.
/u <url>
Launches your default web browser with the given URL.
/f <search string>
Highlights the last occourrence of the specified search string in the chat window.
